# Documentation Index

This file guides readers through the project documentation package for the Royal Palace Hotel Website.

## Included Documents

- `README.md` — Project overview, repository structure, and quick start instructions
- `ARCHITECTURE.md` — System architecture, topology, and AWS component mapping
- `DEPLOYMENT_GUIDE.md` — End-to-end AWS deployment steps for S3 and CloudFront
- `LIVE_DEMO.md` — Live deployment summary and evidence images
- `QUICK_REFERENCE.md` — Fast access to deployment commands and operational reminders
- `deployment-info/SERVICES.md` — AWS service inventory and deployment roles
- `deployment-info/COSTS.md` — Cost categories, estimate notes, and cost-saving considerations
- `metrics/PERFORMANCE_STATS.md` — Key performance metrics and monitoring guidance

## Purpose

The package is designed to present the project as a professional AWS deployment portfolio, including secure static hosting, CDN delivery, and measurable performance.

## How to Use

1. Start with `README.md` for the project overview.
2. Review `ARCHITECTURE.md` for the AWS deployment topology.
3. Follow `DEPLOYMENT_GUIDE.md` for deployment and caching steps.
4. Use `LIVE_DEMO.md` to verify live deployment evidence.
5. Keep `QUICK_REFERENCE.md` handy for deployment commands.
6. Consult `deployment-info/SERVICES.md` and `deployment-info/COSTS.md` for operational details.
7. Reference `metrics/PERFORMANCE_STATS.md` for performance checks.
